### Time complexity ArrayList vs LinkedList
| Operation              | ArrayList | LinkedList |
| :--------------------- | :-------: | :--------: |
| Insertion at ending    |   O(1)    |    O(1)    |
| Insertion at beginning |   O(n)    |    O(1)    |
| Access by index        |   O(1)    |    O(n)    |
| Remove by index        |   O(n)    |    O(n)    |
